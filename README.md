# State Switcher
## 问题
运行过程中会根据逻辑状态给组件的属性给予不同的值

## 解决
通过在Inspector中创建不同的State并且给与组件不同的赋值

## 方法
1. 在根节点添加组件StateSwitcher
2. 点击State List中Add按钮
3. 输入状态名，点击确定
4. 添加编辑选项
